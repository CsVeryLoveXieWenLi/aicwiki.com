# 宝箱

通过在战斗中击杀魔族可以获得宝箱。宝箱的获得位置参见[宝箱地图](/zh/maps)、[战斗点](/zh/battle-locations)。

每次区域危险度被重置后，仅在各个战斗点的首次战斗中可以获得宝箱。诺艾儿战败或返回据点时区域危险度将被重置，同时所有携带的宝箱被开启。

# 效果转轮

通过击破战斗点提高`时间危险度`以及丢弃诺艾儿体液提高`体液危险度`可以获得效果转轮。更多关于危险度的信息参见[战斗点](/zh/battle-locations)。

可以获得的效果转轮数量的计算规则如下：

<details>
  <summary>点击展开</summary>

可以获得的效果转轮数量与通过击破战斗点提高时间危险度的次数（以下简称`有效战斗数`）、`时间危险度`和`体液危险度`有关：

```c
若 (3 < 时间危险度 < 8) 则:
  时间相关数量 = (时间危险度 + 3) / 5
否则:
  时间相关数量 = (时间危险度 + 2) / 5

若 (3 < 体液危险度 / 3 < 8) 则:
  体液相关数量 = (体液危险度 / 3 + 3) / 5
否则:
  体液相关数量 = (体液危险度 / 3 + 2) / 5

可以获得的效果转轮数量 = min(255, min(时间相关数量, 有效战斗数) + min(体液相关数量, 有效战斗数 + 2))
```

其中`时间危险度`没有上限，`体液危险度`的上限为45。所有除法向下取整。

下面举例说明：

<details>
  <summary>点击展开</summary>

```c
// 有效战斗数 = 0; 时间危险度 = 0; 体液危险度 = 45
时间相关数量 = (0 + 2) / 5 = 0
体液相关数量 = (45 / 3 + 2) / 5 = 3
可以获得的效果转轮数量 = min(255, min(0, 0) + min(3, 0 + 2)) = 2

// 有效战斗数 = 3; 时间危险度 = 7; 体液危险度 = 0
时间相关数量 = (7 + 3) / 5 = 2
体液相关数量 = (0 / 3 + 2) / 5 = 0
可以获得的效果转轮数量 = min(255, min(2, 3) + min(0, 3 + 2)) = 2
```

</details>

</details>

可以获得的效果转轮种类的规律如下：

<details>
  <summary>点击展开</summary>

前 3 个为一组同时确定，每个从以下组合中不重复随机抽取一组：
- 品质+ (100%)
- 数量+ (100%)
- 品质++ (25%) / 数量++ (25%) / 金币+ (25%) / 数量+ (25%)

第 4 个恒定为：数量X。

此后每 3 个为一组同时确定，每个从以下组合中不重复随机抽取一组：
- 若已获得效果转轮数 < 7：
  - 福袋 (100%)
  - 金币+ (100%)
  - 品质+ (30%) / 数量+ (70%)
  - 品质++ (24%) / 数量++ (56%) / 品质+ (6%) / 数量+ (14%)
- 若 7 <= 已获得效果转轮数 < 15：
  - 福袋 (50%) / 金币+ (50%)
  - 品质++ (24%) / 数量++ (56%) / 品质+ (6%) / 数量+ (14%)
  - 品质+++ (40%) / 数量+++ (60%)
- 若已获得效果转轮数 >= 15：
  - 福袋 (50%) / 金币+ (50%)
  - 品质++ (24%) / 数量++ (56%) / 福袋 (6%) / 数量+ (14%)
  - 品质+++ (40%) / 数量+++ (60%)
  - 品质+++ (30%) / 数量+++ (70%)
  - 数量X (30%) / 数量+++ (70%)

注意，已获得效果转轮数不同于当前携带效果转轮数。下一组会在当前已确定的效果转轮都已被获得时（而不是将要获得下一组的第一个效果转轮时）确定。<span class="spoiler">因此可以通过加载获得每组最后一个效果转轮前的存档来改变下一组的随机数。</span>

</details>

效果转轮在采集素材或开启宝箱时将按照获得顺序被使用，且每次最多使用 8 个。若携带的效果转轮数大于 8 个，则采集素材时将使用最后获得的 8 个，开启宝箱时将被要求丢弃多出的数量。每次采集素材或开启宝箱时能够获得的物品数量上限为 99。诺艾儿战败强制开启宝箱时无法使用效果转轮。携带的效果转轮将在区域危险度重置时被清空。
